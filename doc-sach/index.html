Cài giọng đọc tiếng Việt

Trình duyệt của bạn đang đọc bằng giọng mặc định (có thể không phải tiếng Việt) vì máy chưa cài giọng đọc tiếng Việt hệ thống.

Windows 10/11: Settings → Time & Language → Speech → mục "Manage voices" → Add voices → chọn Vietnamese (Việt Nam) → tải về. Sau đó tải lại trang này.

macOS: System Settings → Accessibility → Spoken Content → System voice → tải giọng "Linh" (Vietnamese).

Điện thoại Android/Chrome: thường đã có sẵn giọng "Tiếng Việt" — chọn trong ô chọn giọng ở thanh công cụ.
